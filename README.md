# Nexacro_Study

항목 (Item),역할 (Role)
Environment,애플리케이션의 전역 설정을 관리하는 최상위 항목입니다.
ScreenDefinition,다양한 해상도/장치에 따른 화면 크기 정의 및 설정을 관리합니다.
Desktop_screen,데스크톱 환경 등의 특정 화면에 대한 정의를 담고 있습니다.
Variables (Screen Definition 하위),해당 화면(Screen) 내에서 사용되는 전역 변수를 관리합니다.
Cookies,애플리케이션에서 사용되는 쿠키 관련 설정을 관리합니다.
HTTP Header,서버 통신 시 사용되는 HTTP 헤더 설정을 관리합니다.
Script,전역 JavaScript 스크립트 파일 및 설정을 관리합니다.
TypeDefinition,"넥사크로 N 내에서 사용되는 사용자 정의 타입 (예: Objects, Services) 등을 정의합니다."
Objects,공통 컴포넌트나 재사용 가능한 기능을 정의하는 항목입니다.
Services,"서버와 통신하기 위한 서비스 (URL, 프로토콜 등)를 정의합니다."
ProtocolAdaptors,통신 프로토콜을 처리하는 어댑터 정의를 관리합니다.
DeviceAdaptors,"장치별 기능 (예: 카메라, GPS)을 활용하기 위한 어댑터를 관리합니다."
CordovaPlugins,하이브리드 앱 개발 시 사용되는 Cordova 플러그인 설정을 관리합니다.
Application Information,애플리케이션의 전반적인 정보와 설정을 관리합니다.
Application Variables,애플리케이션 전역 변수 (데이터셋/변수)를 관리합니다.
Datasets,애플리케이션 전체에서 공유되는 공통 데이터셋을 정의합니다.
Variables (Application Information 하위),애플리케이션 전체에서 공유되는 공통 변수를 정의합니다.
DataObjects,데이터 구조 정의 및 관련 설정을 관리합니다.
Applications,메인 애플리케이션 파일 (예: Application_Desktop)을 관리합니다.
Base / FrameBase,기본 프레임워크 또는 공통 모듈/폼 등의 기반 구조를 관리합니다.
